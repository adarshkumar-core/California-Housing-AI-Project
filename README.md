# California Housing Price Prediction using Machine Learning

This project predicts housing prices using machine learning models based on the California Housing dataset.

## 📌 Overview
The goal is to build an AI model that predicts house prices using various features like income, population, location, and housing characteristics.

## 🚀 Features
- Data preprocessing and visualization
- Model training using Random Forest
- Model comparison with Linear Regression
- Performance evaluation using MAE and RMSE

## 🧠 Algorithm Used
- Random Forest Regressor (Primary Model)
- Linear Regression (Comparison Model)

## 📊 Results
Random Forest achieved lower RMSE compared to Linear Regression, making it the better model for prediction.

## 🛠 Tools Used
- Python
- Google Colab
- Scikit-learn
- Matplotlib / Seaborn
